# World Layoffs SQL Data Analysis Project

## Project Overview

This project focuses on cleaning and analyzing a real-world layoffs dataset using SQL.

The project is divided into two major phases:

1. Data Cleaning
2. Exploratory Data Analysis (EDA)

The goal of this project is to practice real-world SQL data analyst workflows including:
- Data cleaning
- Data transformation
- Handling duplicates
- Standardizing inconsistent data
- Exploratory business analysis
- Trend analysis using SQL

---

## Dataset Information

Dataset: Layoffs Dataset

The dataset contains information about:
- Company layoffs
- Industry
- Country
- Funding raised
- Company stage
- Total employees laid off
- Percentage layoffs
- Dates of layoffs
